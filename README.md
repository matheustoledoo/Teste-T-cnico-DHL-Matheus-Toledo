# Painel Operacional – ZYX Logística  
### Teste Técnico – Analista de Sistemas Operacionais JR – DHL - Matheus Roveri Toledo

Este projeto foi desenvolvido como solução para o desafio técnico da DHL Supply Chain, demonstrando habilidades de construção de uma aplicação web simples, clara e funcional para apoiar processos operacionais.

A aplicação simula um Painel Operacional, permitindo visualizar, cadastrar e atualizar operações logísticas, representando um fluxo básico de controle dentro de um ambiente de warehouse ou transporte.

---

## Tecnologias Utilizadas

### **Backend**
- Node.js  
- Express  
- CORS  
- Arquitetura separando responsabilidades (server + repository)

### **Frontend**
- HTML  
- CSS  
- JavaScript puro (Fetch API para consumo do backend)

---

## Funcionalidades

### Listagem de operações
Cada operação contém:
- Código  
- Cliente  
- Origem  
- Destino  
- Status  
- Última atualização  

### Filtro por status  
Permite exibir apenas operações:
- Pendente Coleta  
- Em Trânsito  
- Entregue  

### Cadastro de novas operações  
Formulário simples integrado ao backend.

### Atualização de status  
Direto pela tabela.
